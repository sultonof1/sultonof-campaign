# Sultonof Campaign Website 🚀

To'liq ko'p tilli (UZ/RU/EN) kampaniya boshqaruv saytining loyihasi.

## Xususiyatlari ✨

- ✅ 3 tilda (Uzbek, Russian, English) to'liq lokalizatsiya
- ✅ Admin Panel (Login/Parol bilan himoyalangan)
- ✅ Xodimlar boshqaruvi + Rasm + Statistika
- ✅ Kompaniya ma'lumoti va galereyasi
- ✅ Portfolio/Proyektlar boshqaruvi
- ✅ Rasm preview va yuklash
- ✅ Modal tasdiqlash (O'chirish)
- ✅ Search va Sorting
- ✅ Light Mode (Pastel ranglar)
- ✅ Responsiv dizayn (Mobile, Tablet, Desktop)

## Stack 🛠️

**Backend:**
- Node.js + Express
- MongoDB
- JWT Authentication
- Multer (Rasm yuklash)

**Frontend:**
- React 18
- React-i18n (Multi-language)
- Axios
- CSS3 (Light Mode)

## O'rnatish 📦

### Backend O'rnatish
```bash
cd backend
npm install
npm start
```

### Frontend O'rnatish
```bash
cd frontend
npm install
npm start
```
